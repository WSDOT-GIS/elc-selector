Route locator demo

This page demonstrates the route locator control.

Adding routes

  • Click on a state route
  • If you clicked near a route a point will be added to the map. Otherwise an error message will be displayed.
  • If you have two points along the same route you can double click to create a route segment connecting the two points.

Removing Routes

  • You can click the Delete button to delete the last point that was added to the map.
  • You can click the Clear button to delete all points from the map.
